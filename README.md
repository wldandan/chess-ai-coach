# ♟️ Chess Coach - AI 国际象棋复盘 Chrome 插件

> 给 14 岁孩子的趣味国际象棋复盘工具 🏆⭐🎮

## 命名说明

- **Chess Coach**：国际象棋教练，一个有趣的国际象棋复盘 Chrome 插件
- **chess-analyst Agent**：OpenClaw 中的 AI 复盘分析专家 Agent

## 功能特色

- 🎯 **智能复盘** - 一键分析 chess.com 对局，发现漏着与妙着
- 🏆 **游戏化系统** - XP 经验值、称号成就、雷达图能力分析
- 💬 **趣味评语** - AI 生成符合青少年口吻的复盘解说
- ⚡ **快速响应** - 插件即点即用，无需切换页面

## Agent 分工

| Agent | 职责 | 输入 | 输出 |
|-------|------|------|------|
| `crawler` | 抓取 chess.com 对局数据 | 用户名/游戏链接 | PGN 棋谱 |
| `analyzer` | Stockfish 引擎分析 | PGN 棋谱 | 漏着/妙着列表 |
| `chess-analyst` | AI 深度复盘分析 | PGN + 分析结果 | 结构化复盘报告 |
| `reviewer` | 生成趣味复盘卡片 | 分析结果 | 中文复盘报告 |
| `gamification` | 游戏化数据处理 | 复盘报告 | XP/称号/成就 |

## chess-analyst Agent

**角色**：耐心的国际象棋教练，专门帮助 14 岁左右的孩子提高棋艺。

**核心能力**：
- 📊 棋局概览（结果、回合数、时间控制）
- 🎯 亮点时刻（值得肯定的着法）
- ⚠️ 关键失误（按重要性排序，含正确着法建议）
- 💡 可优化之处
- 📚 开局学习建议
- 🌟 今日收获

**触发词**：`帮我分析` / `复盘` / `这盘棋` / `chess analysis`
